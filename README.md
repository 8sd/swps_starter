# SWPS Example

This is an example project that shows how Vue + Nuxt + Vuetify can be used together.

## How to Run the Code

### Setup

Make sure to install the dependencies:

```bash
yarn install
```

### Development Server

Start the development server on `http://localhost:3000` run:

```bash
yarn dev -o
```
